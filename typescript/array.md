## 如何设置数组和数组元素同时不可以改？
```
 const arr = [1,2,3] as const; //1. 用const定义 2. as const 
```
## 可变元组
* 正常的元组是固定的，类型和个数都是固定的；
```
const arr:[number,string] = [1,'2'];
```
* 可变元组，可以定义数组的个数是变化的,可以用`···`接收；
```
const arr: [number,...string[]] = [1,'2','3']; //用`···`定义数组个数是变化的；

//解构数组
let [a,b,...c] = arr; // a=1 b='2' c=['3']
```
